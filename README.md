## Install

```bash
conda env create --file environment.yml
source activate cv
conda install -n cv python=3.6
pip install -r requirements.txt
```

## 帧结构

| Name | Length | Description   | Value(DEC) |
| ---- | ------ | ------------- | ---------- |
| SOF  | 8      | 起始段        | 87         |
| ID   | 4      | 数据ID        | 0-15       |
| LEN  | 4      | 数据长度n     | 0-15       |
| DATA | 16n    | uint16        |            |
| CRC8 | 8      | 检验段 |            |
| EOF  | 8      | 结束段        | 88         |
